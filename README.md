# Python Web Scraper & Alerts System

An automated Python application that scrapes websites daily, parses relevant data (e.g. prices), stores historical results, and sends email alerts when specified conditions are met.

## Features
- Scheduled web scraping
- Dynamic HTML parsing
- JSON data storage
- Automated email notifications

## Tech Stack
- Python
- Requests
- BeautifulSoup
- SMTP
- Cron / Task Scheduler
